1. Load account information from file
2. Repeatedly get an account number, PIN and option, and process option.
3. Dump updated account information to file.

Refinement of 2:

For each account number:
    If the account exists:
        2.1 Get the PIN
        If the PIN is correct:
            2.2 Display options
            2.3 Get option
            2.4 Process option
        Else:
            2.5 Report incorrect PIN
    Else:
        2.6 Report invalid account number

Refinement of 2.4:

If option is not valid:
    2.4.1 Warn user and skip
Else:  
    If option is w:
        2.4.2 Handle withdrawl
    Else if option is d:
        2.4.3 Handle deposit
    Else if option is b:
        2.4.4 Print current balence
    Else if option is m:
        2.4.5 Handle history dump
    Else if option is c:
        2.4.6 Inform user of cancellation

Refinement of 2.4.2:

2.4.2.1 Get withdrawl amount
If withdrawl amount is invalid or greater than current balence:
    2.4.2.2 Warn user and skip
Else:
    2.4.2.3 Reduce balence
    2.4.2.4 Update history

Refinement of 2.4.3:

2.4.3.1 Get deposit amount
If amount is invalid:
    2.4.3.2 Warn user and skip
2.4.3.3 Increase balence
2.4.3.4 Update history

Refinement of 2.4.5:

For latest 6 transactions in history:
    2.4.5.1 Get date of transaction
    2.4.5.2 Get time of transaction
    2.4.5.3 Get type of transcation
    2.4.5.4 Get value of transaction
    2.4.5.5 Print this data in the form "date, time, type, value"
